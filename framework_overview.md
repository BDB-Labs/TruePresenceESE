# ESE Framework Overview

Ensemble Software Engineering (ESE) is a lightweight, reproducible workflow for AI-assisted development.

**Core idea:** different intelligences take different jobs.

## Goals

- reduce self-confirming reasoning
- create structured disagreement
- keep humans in charge of risk and merging
- produce auditable artifacts at each stage

## Roles

- Architect
- Implementer
- Adversarial Reviewer
- Security Auditor
- Test Generator
- Performance Analyst

## Human authority

Humans own scope, risk acceptance, and final merge decisions. AI provides decision-support, not governance.
